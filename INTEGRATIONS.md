# Connected Integrations

These services are connected via MCP. ALWAYS use the MCP tools to interact with them.
NEVER use local apps (Mail.app, Calendar.app), AppleScript, osascript, or shell commands for these services.

## Local Computer
Files, folders, and system access on this Mac
MCP tools: mcp__filesystem__*

## Google Workspace
Gmail, Calendar, Sheets, Drive
MCP tools: mcp__gmail__*, mcp__google-calendar__*, mcp__google-sheets__*

## GitHub
Repos, issues, pull requests, code search
MCP tools: mcp__github__*

## Figma
Design files, components, comments
MCP tools: mcp__figma__*

## Airtable
Bases, tables, records, views
MCP tools: mcp__airtable__*

## Apple Apps
Reminders, Notes, Calendar, Contacts, Mail (macOS)
MCP tools: mcp__apple__*

## Salesforce
CRM, leads, opportunities, accounts, reports
MCP tools: mcp__salesforce__*
